## 文档规范模板详见：[文档规范模板.md][1]

**本项目使用Markdown来编辑翻译的文档等资料，请按Markdown格式编写自己负责翻译的部分：**


- **1.** 翻译过程中请译者使用通俗易懂的文字，不必逐字逐句翻译；
- **2.** 如需另加注解，请用小括号提醒。如：DST（*文档规范模板，译者注*）；
- **3.** 请在章节末尾附加译者的基本信息；
- **4.** 未尽事宜，请关注更新。
